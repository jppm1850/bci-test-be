# bci-test-be

- Aplicación Spring Boot para guardar información de usuarios y generar alertas por medio de Amazon SQS desencadenando lambda

# Técnologias usadas

* Spring boot 2.4
* RxJava
* Java 11
* Bean Validation
* Spring WebFlux
* AWS SQS
* AWS Lambda
* NodeJS
